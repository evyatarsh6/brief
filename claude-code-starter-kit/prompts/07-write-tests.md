# Prompt: Write Tests

```text
Add or improve tests for <FEATURE_OR_BUG>.

Instructions:
1. Inspect existing test patterns first.
2. Prefer behavior-based tests.
3. Cover happy path, failure path, edge cases, and regression cases.
4. Do not weaken or delete existing tests.
5. If code is hard to test, explain why before changing production code.
6. Run the relevant test command and report the result.
```
