# Prompt: Start New Project

```text
I want to start a new software project using Claude Code.

Before writing any code:
1. Interview me with focused questions.
2. Identify users, goals, constraints, integrations, risks, MVP scope, and open questions.
3. Separate facts from assumptions.
4. Do not suggest architecture yet unless it is required to ask a better question.
5. At the end, summarize what we know and what is still missing.
```
