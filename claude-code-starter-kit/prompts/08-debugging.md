# Prompt: Debugging

```text
Investigate this bug: <BUG_DESCRIPTION>.

Do not edit code first.

Return:
1. Observed behavior.
2. Expected behavior.
3. Relevant files and execution path.
4. Evidence from code/logs/tests.
5. Ranked hypotheses.
6. Minimal fix plan.
7. Regression test plan.

Only after the investigation, ask for approval to implement the fix.
```
