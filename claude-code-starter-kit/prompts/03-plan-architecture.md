# Prompt: Plan Architecture

```text
Based on docs/PRD.md, propose an architecture for this project.

Before recommending one design:
1. Present 2-3 viable architecture options.
2. Compare them by complexity, maintainability, scalability, testability, security, observability, and delivery speed.
3. Recommend the best option for MVP.
4. Explain the path to production hardening.
5. Create or update docs/ARCHITECTURE.md only after the plan is clear.
6. Do not write implementation code yet.
```
